📁 檔案完整性 4 項
1
單一 HTML 檔
只有一個 .html，無外掛 CSS/JS 檔案依賴
✅ ls -lh
2
檔案大小合理
20–80 KB（純 HTML+CSS+JS，無 base64）
ls -lh file.html
3
行數合理
500–1500 行（太少=功能不足，太多=有問題）
wc -l file.html

🔤 字體系統 5 項
7
3 個 Google Fonts
Playfair Display + Inter + Noto Sans SC
grep -c fonts.googleapis.com
8
Display 字體用於標題
font-family: 'Playfair Display' 用於大標/章節標
grep 'Playfair Display'
9
Heading 字體用於中層
font-family: 'Inter' 用於卡片標題/流程標題
grep "'Inter'"
10
Body 字體用於內文
font-family: 'Noto Sans SC' 用於內文段落
grep 'Noto Sans SC'
11
無 PPT 殘留字體
無 +mn-ea、Calibri、Microsoft YaHei
grep -ciE 'calibri|yahei|mn-ea'
🎨 CSS 架構 5 項
12
CSS 變數 ≥ 20
:root 內有 ≥ 20 個 --name: value
grep -oE '--[a-z][a-z0-9-]*:' | sort -u | wc -l
13
CSS 模組化註解
有 /* ─── */ 或 /* ═══ */ 區段標題 ≥ 8 個
grep -c '/\*[-═]'
14
8px base spacing grid
--s1 到 --s7 存在且為 8 的倍數
grep -oE '--s[1-7]: [0-9]+px'
15
設計代幣分類
有色彩、字體、間距、動畫時間等分類註解

16
