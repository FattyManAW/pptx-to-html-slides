S5
架構決策記錄
D1 Spec Review + D2 Token Table 中做出的關鍵決策

Decision 1

--ds-* 為 Design System 單一事實來源

三產品 accent 不直接用 --accent / --accent-2，而是經由 --ds-impacts-brand / --ds-aps-violet / --ds-cris-carbon 抽象層。產品切換只改一處。

✅ Nana 可確認

Decision 2

APS --c-t* → --t* 映射

APS 的 --c-t1~c-t4 是 text-level tokens，與通用 --t1~t4 語意相同但命名不同。映射策略：--c-t1→--t1，保持 APS CSS 不破壞，新增通用 canonical。

✅ D2 已完成

Decision 3

APS --c-* → --ink-* 映射

--c-600→--ink-500, --c-700→--ink-700, --c-800→--ink-800, --c-900→--ink-900, --c-950→--ink-950。APS 色階與通用 ink-* 不連續，就近映射。

✅ D2 已完成

Decision 4

APS accent-extensions 保留為 APS-only

--accent-amber / --accent-blue / --accent-em / --accent-rose 為 APS 四色擴展。目前無對應润思/CRIS 版本，不強制納入 canonical。

⚠️ 待 Nana 確認

Decision 5

--blur-* 小寫命名統一

APS --blur-sm/md/lg/xl → 通用 --blur-8/12/24/32。數字化命名消除 "sm/md/lg/xl" 語意模糊。

✅ D2 已完成

Decision 6

--r-* → border-radius 值

--r-sm/md/lg/xl 映射到固定 px 值（6/12/16/24）。APS 無對應通用 token，此為 token→value 映射而非 token→token。

✅ D2 已完成
