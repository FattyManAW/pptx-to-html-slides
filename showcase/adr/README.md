## ADR 格式

每篇 ADR 遵循 [Michael Nygard 格式](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)：

```
Context  → 當時的情境與約束
Decision → 做了什麼決定 + 被拒絕的替代方案
Consequences → 正面影響 + 取捨 + 未來方向
Status   → proposed | accepted | deprecated | superseded
```
